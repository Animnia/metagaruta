# Metagaruta Online

**オンライン競技かるた —— 在线竞技歌牌游戏**

## 游戏网址：<https://metagaruta.com>

Metagaruta 是一款多人在线竞技歌牌（かるた）游戏。玩家通过聆听音乐片段，在场上的歌牌中抢先找到对应曲目，以此竞争得分。项目名取自日语「メタガルタ」（Meta + Garuta），寓意融合了互联网时代元素的新型歌牌玩法。

## 🎮 游戏玩法

1. 创建或加入一个房间（房间号为 4 位数字）
2. 房间内所有非房主玩家点击「准备」，房主点击「开始游戏」
3. 游戏开始后，界面上展示 **16 张歌牌**（来自 25 首曲目的题库池）
4. 每轮播放一段音乐片段（最长 45 秒），玩家需要：
   - **抢答正确歌牌** → **+10 分**
   - **答错** → **-5 分**，且本轮不可再操作
   - 当前曲目可能 **不在场上**，此时可点击「没有这首歌」→ **+5 分**
5. 当 16 张歌牌全部被正确匹配后，游戏结束，展示最终排名

### Vocaloid 模式

歌牌上显示歌曲名称（支持切换原文/译文），玩家根据音乐片段识别对应歌曲。

### 东方 Project 模式

歌牌上显示角色立绘图片，玩家根据音乐片段识别对应的东方角色。可在设置中选择是否显示角色名称。
